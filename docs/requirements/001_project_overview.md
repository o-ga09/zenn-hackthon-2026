# 旅行Vlog自動生成エージェント - プロジェクト概要

**プロジェクト名:** AI Travel Vlog Generator
**バージョン:** 1.0
**作成日:** 2026年1月6日
**ハッカソン:** 第4回 Agentic AI Hackathon with Google Cloud

---

## 1. プロジェクト概要

### 1.1 目的
ユーザーがアップロードした旅行の画像・動画から、AIエージェントが自律的にずんだモンの声で魅力的なショート動画（Vlog）を生成するWebアプリケーション。

### 1.2 コンセプト
「旅行の振り返りを AI で、簡単に、便利に、そして、もっとエモく」

### 1.3 ターゲットユーザー
- 旅行好きで動画編集の知識が少ない人
- SNSに素早くコンテンツを投稿したい人
