## 11. リスク管理

| リスク | 影響度 | 対策 |
|--------|--------|------|
| VOICEVOX統合の失敗 | 高 | 代替TTS（Google Cloud TTS）を準備 |
| Vertex AI制限超過 | 中 | リクエスト制限監視、バックオフ実装 |
| FFmpeg処理タイムアウト | 中 | タイムアウト設定、処理の分割 |
| Cloud Run冷起動遅延 | 低 | min-instancesを1に設定（コスト考慮） |
| 開発期間不足 | 中 | MVP機能に集中、Phase 2は後回し |
